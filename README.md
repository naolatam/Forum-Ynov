# Warframe forum project
By Silk Road

## 📚 Description
This project is a forum based on the video game named Warframe, featuring moderation roles (as admin, moderator or a user, registered or not), live notification system and other elements which will be described in this file.

## ✨ Features
- ***HTTPS protocol***: The forum uses a Secure Sockets Layer (SSL). A self-signed certificate is generated by the program if none exists.
- ***Authentification page*** : you can be registered on the forum via OAuth login, using a Google/GitHub account.
- ***Post management*** : on the forum you can manage your post by editing it or simply delete it.
- ***Pictures supporter*** : bring life to your post by adding a picture (JPEG, PNG, GIF).
- ***Moderation roles*** : you can be an admin, a moderator, a registered user or a simple guest. Those roles will determine your possibilities on the forum.
- ***Like/Dislike*** : as a registered user, you can like or dislike a post and/or comments.
- ***Comment*** : as a registered user, you can comment a post.
- ***Live Notification System***: Registered users can receive notifications and view a "Recent Activity" section on their profile page, displaying the latest notifications. Additionally, notifications are sent via email.
- ***Admin Panel***: A dedicated admin panel allows administrators to manage categories (create, update, delete), assign user roles (e.g., user to moderator), and moderate content by validating new posts and reviewing reported ones.

## ▶️ How to Run
### With Docker
1) Ensure Docker and Docker Compose are installed on your system.
2) Clone the repository from GitHub using this command: 
    ```bash
    git clone https://github.com/naolatam/Forum-ynov
    ```
3) Navigate to the project directory: `cd Forum-ynov/code`
4) Copy the `.env.example` file from `Forum-ynov/code/src` folder to `.env` and complete it with the required configuration values.
    - Google auth client credentials for Oauth can be generated [here](https://console.cloud.google.com/auth/clients)
    - Github auth client credentials for Oauth can be generated [here](https://github.com/settings/developers)
5) Build and start the containers using the command: `docker-compose up --build`
6) Open your web browser and go to: `https://localhost:8080`

### Without docker
1) Clone the Repository on GitHub, using this link :
2) Navigate to the project directory : `Forum-ynov/code/src/`
4) Copy the `.env.example` file from `Forum-ynov/code/src` folder to `.env` and complete it with the required configuration values.
    - Google auth client credentials for Oauth can be generated [here](https://console.cloud.google.com/auth/clients)
    - Github auth client credentials for Oauth can be generated [here](https://github.com/settings/developers)

3) use the command : `go run ./cmd`
4) go on your web browser on this address : `https://localhost:8080`
